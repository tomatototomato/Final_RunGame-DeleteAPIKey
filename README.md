# コイン集めゲーム (Coin Collect Game)

UnityとFirebaseを連携させた、オンラインランキング機能付きの3Dランアクションゲームです。

## 1. ゲーム概要

制限時間内に流れてくるコインを集め、ハイスコアを目指すスコアアタック型の3Dランアクションゲームです。シンプルな操作ながら、障害物を避ける判断力とスコア獲得の正確さが求められます。

### ゲームシステム

- **スコア獲得**: 画面奥から流れてくる「金・銀・銅」のコインに触れることでポイントが加算されます。
- **障害物**: コインと同様に奥から流れてくる障害物を避ける必要があります。
- **制限時間**: 限られた時間内でいかに効率よくスコアを稼げるかを競います。

### 操作方法

- **移動**: マウスでクリックした地点にキャラクターが追従して移動します。
- **ジャンプ**: スペースキーを押すと、アニメーション付きのジャンプを行います。

---

## 2. オンライン機能 (Firebase連携)

Googleのバックエンドサービス「Firebase」を活用し、ユーザー認証とリアルタイムランキングを実装しています。

### アカウント管理 (Firebase Authentication)

- **サインアップ**: メールアドレスとパスワードによる新規登録が可能です。
- **メール認証**: 不正登録防止のため、登録後に送信される確認メール内のリンクをクリックすることでアカウントが有効化されます。
- **セキュリティ**: メール認証が完了していないユーザーはゲーム画面へ遷移できない制限を設けています。

### ランキング機能 (Cloud Firestore)

- **リアルタイム集計**: クラウドデータベースを利用し、全プレイヤーのスコアをリアルタイムで集計します。
- **トップ10表示**: スコアの高い順に上位10名のランキングを表示します。
- **スコア反映**: 自分の最新スコアを即座にデータベースへ反映し、他プレイヤーと競い合うことができます。

## 4. 導入・実行に関する注意点

### 設定ファイルの除外について

セキュリティ保護のため、Firebaseとの接続に必要な **`google-services.json` は本リポジトリから除外しています。**

本プロジェクトを自身の環境で動作させる場合は、以下の手順が必要です。

1. [Firebase Console](https://console.firebase.google.com/) で自身のプロジェクトを作成する。
2. Unityアプリをプロジェクトに登録し、発行された `google-services.json` を取得する。
3. 取得したファイルを、Unityプロジェクト内の `Assets` フォルダ直下に配置する。

### 実行方法

1. Unityエディタでプロジェクトを開きます。
2. `Scenes/LoginScene` を開き、再生ボタンを押して開始します。
3. 初めての場合はサインアップを行い、届いたメールで認証を済ませてからログインしてください。
